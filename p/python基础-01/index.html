Python变量简介

变量的概念

变量源自数学，是计算机语言中用于储存计算结果或表示值的抽象概念，可被多次赋值并始终保留最新值。其命名规则为：仅能由字母、数字和下划线组合，首字符不能是数字，区分大小写，且不能使用特殊关键字。

变量赋值

在Python中，变量无需声明，必须先赋值后使用，赋值时才会创建变量；变量本身没有类型，我们所说的“类型”是指变量所指向的内存中对象的类型。

2

x = 3  # int
x = 3.0  # float
x = 'three'  # string
x = True  # bool

在Python中，等号’=‘是赋值语句，可将任意数据类型赋值给变量，同一个变量可反复赋值且能是不同类型；变量本身类型不固定的语言叫动态语言，与之对应的静态语言（如Java）定义变量时必须指定类型，赋值时类型不匹配会报错。

常见变量类型

  • int (整型)
    定义：即整数，包含正整数和负整数，不带小数点。
    特性：Python 3 中的整型没有大小限制（相当于 Python 2 的 Long 类型），因此Python 3 不再区分 int 和 long。
  • float (浮点型)
    定义：由整数部分与小数部分组成。
    表示法：支持科学计数法 (如$2.5e2 = 2.5 \times 10^2 = 250$)。
  • string(字符串)
    定义：使用单引号、双引号、三单引号或三双引号创建，三引号支持多行字符串。
    特性：Python 无单字符类型，单字符也作为字符串处理。

常见字符串操作

切片

在字符串切片操作中，使用 s[开始索引:结束索引:步长] 的格式进行提取，遵循“包头不包尾”的原则，即包含开始索引对应的字符但不包含结束索引对应的字符，且步长默认为1。索引从0开始计数，若为负数则表示从字符串末尾倒数，其中-1代表最后一个字符。此外，支持省略写法：省略开始索引表示从头部开始，省略结束索引表示一直截取到尾部，而将步长设为-1则可直接实现字符串的逆序。

print(str[::-1])  # 将str反向

查找

字符串的 find() 方法主要用于检测一个字符串中是否包含指定的子字符串。如果找到了子字符串，它会返回子字符串首次出现的索引值；如果没找到，则返回 -1。

使用时可以指定查找的起始位置（beg）和结束位置（end）来限定范围，如果不指定这两个参数，默认会在整个字符串范围内进行查找。简单来说，它的逻辑就是：找到返回索引，找不到返回 -1。

Python数据结构简介

数据结构的核心在于解决批量数据的组织问题，它避免了为每个数据单独定义变量（如处理100个数据需100个变量）的低效做法，而是将大量数据有机组织起来，方便编程时统一操作。不同的数据组织方式形成了多样的数据结构，每种结构都有自身特点和适用场景，以此提升数据处理的效率和便捷性。

Tuple(元组)

元组本质上也是一种序列，但它与列表最大的区别在于不可修改，这一点和字符串类似。

在创建元组时，最核心的语法是逗号，只要用逗号分隔多个值，Python就会自动将其识别为元组。虽然大部分时候我们会用圆括号()把元素括起来，但这并非强制要求。

关于创建元组，有几个特殊的细节需要注意：

  • 如果想创建一个空元组，直接用空的圆括号()即可。
  • 如果元组里只有一个元素，必须在元素后面加一个逗号，例如t5=(1,)，否则Python会把它当作普通的变量处理。
1
2
3

l = 'Python'
t1 = tuple(l)  # 使用tuple()创建元组
t2 = (1,2,3)  # 直接创建一个元组

List(列表)

列表是Python中一种重要的数据结构，它由写在方括号[]内、并用逗号分隔的一系列元素组成。列表具有极大的灵活性，其核心特点包括：

  • 列表中的元素类型可以不相同，既可以是数字、字符串，甚至可以包含其他列表，即支持嵌套。
  • 列表是可变的，这意味着列表中的元素可以被修改、添加或删除，这是它与字符串和元组最根本的区别。

Set(集合)

集合是Python中一种无序且不包含重复元素的数据结构，它由写在大括号{}内、用逗号分隔的一系列元素组成，或者通过set()函数创建。其核心特点包括：

  • 集合中的元素必须是不可变类型（如数字、字符串、元组），且集合本身是无序的，这意味着集合不支持索引或切片操作。
  • 集合是可变的，这意味着集合中的元素可以被添加或删除，这是它与元组最根本的区别。注意：创建一个空集合必须用set()而不是{}，因为{}是用来创建一个空字典。

Dictionary(字典)

字典是Python中唯一的内置映射类型，也常被称为散列表，它由写在大括号{}内、用逗号分隔的一系列键值对组成。其核心特点包括：

  • 字典中的每个元素都有一个对应的“键”（Key），键必须是不可变类型（如数字、字符串、元组），且在同一个字典中不能重复。
  • 字典是可变的，这意味着字典中的键值对可以被修改、添加或删除，这是它与元组最根本的区别。注意：创建一个空字典必须用{}。

Python特殊语句简介

条件结构

条件结构（也叫选择结构或分支结构）是编程中控制程序流程的核心逻辑，它允许代码根据特定条件的真假来决定执行哪一部分逻辑。

条件结构简介

在 Python 中，条件结构主要通过 if、elif（else if 的缩写）和 else 关键字来实现。

  • 基本逻辑：如果（if）某个条件成立，就执行代码块 A；否则（else），执行代码块 B。
  • 语法特点：Python 使用缩进（通常是 4 个空格）来划分代码块，而不是像其他语言那样使用大括号 {}。
  • 常用比较运算符：
    • == (等于)
    • != (不等于)
    • > (大于), < (小于)
    • >= (大于等于), <= (小于等于)

逻辑运算符与“惰性求值”

在 Python 的条件结构中，我们经常需要组合多个条件，这时就需要用到逻辑运算符：and、or 和 not。

  • and (与)：只有当所有条件都为真（True）时，结果才为真。
  • or (或)：只要任意一个条件为真（True），结果就为真。
  • not (非)：对条件取反，真变假，假变真。

惰性求值

Python 在处理逻辑运算符时采用惰性求值（也叫短路求值）策略，这意味着解释器会从左到右评估表达式，一旦结果确定，就会立即停止计算剩余部分，以提高效率。

  • and 的惰性：如果第一个条件为假（False），Python 知道整个表达式肯定为假，因此不会再去检查后面的条件。
  • or 的惰性：如果第一个条件为真（True），Python 知道整个表达式肯定为真，因此不会再去检查后面的条件。

循环结构

循环结构是编程中用于处理重复性任务的核心逻辑，它允许代码根据特定条件或遍历序列来反复执行某一段程序块，从而极大地提高了代码的效率和简洁性。

循环结构简介

在 Python 中，循环主要通过 for 循环和 while 循环两种形式来实现，它们分别适用于不同的场景。

  • 基本逻辑：只要满足设定的条件（while）或者序列中还有元素未遍历（for），就重复执行缩进的代码块。
  • 语法特点：与条件结构一样，Python 使用缩进来划分循环体，且通常配合 break（跳出整个循环）和 continue（跳过本次循环）来控制流程。

For 循环（遍历循环）

for 循环主要用于遍历序列（如列表、字符串、元组）或其他可迭代对象。当你明确知道需要处理哪些数据，或者需要按顺序访问集合中的每个元素时，for 循环是首选。

  • 适用场景：已知循环次数，或需要遍历数据结构中的所有元素。
  • 核心机制：自动从序列中取出元素赋值给变量，直到序列为空。

numbers = [1, 2, 3, 4, 5]
total = 0

# 遍历列表中的每一个数字
for num in numbers:
    total += num  # 累加
    if total > 5:
        # 如果总和超过5，演示 break：直接跳出整个循环
        break 

print(f"最终的总和是: {total}")

While 循环（条件循环）

while 循环则是在条件为真（True）时不断重复执行代码块。它更关注“何时停止”，而不是“遍历什么”。

  • 适用场景：未知循环次数，仅知道结束条件的场景（如用户验证、游戏主循环）。
  • 核心机制：每次执行前判断条件，若条件始终为真且内部没有改变条件的语句，极易导致死循环。

# 只要 count 小于 5，就一直执行

    count += 1
    
    if count == 3:
        # 当数字是3时，演示 continue：跳过本次打印，直接进入下一次循环
        continue 
        
    print(f"当前计数: {count}")

循环控制

在循环执行过程中，我们经常需要根据具体情况调整流程，这时就需要用到 break 和 continue。

  • break (终止)：用于完全跳出当前所在的循环结构。一旦执行 break，循环立即结束，程序继续执行循环后面的代码。常用于找到目标后提前退出，以节省资源。
  • continue (跳过)：用于跳过当前这一次循环中剩余的代码，直接进入下一次循环的判断或取值。它不会终止整个循环，只是忽略当下的这一轮操作。
