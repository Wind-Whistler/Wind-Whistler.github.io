为什么需要数据可视化

我们之所以需要数据可视化，是因为人脑处理视觉信息的速度远超文字和数字。在大数据时代，面对海量的原始数据，单纯看表格很难快速发现其中的规律。数据可视化通过将枯燥的数字转化为直观的图表（如折线图、热力图等），能帮助我们迅速识别趋势、发现异常值并理清复杂的数据关系，从而极大地降低理解成本，辅助我们做出更准确、高效的决策。

Python可视化工具

Matplotlib 则是 Python 生态中最经典、应用最广泛的 2D 绘图库。它设计初衷是为了让 Python 拥有类似 MATLAB 的强大绘图能力，支持生成出版级质量的静态、动态及交互式图表。作为 Python 数据科学生态的基石，Matplotlib 不仅能与 NumPy、Pandas 等数据处理工具无缝配合，还具备极高的可定制性——从基础的折线图到复杂的 3D 图形都能轻松实现，能通过简洁的代码将数据分析结果进行呈现。

注意，由于Matplotlib库不支持中文，所以如果涉及到一些中文标题，请先按照如下代码进行字体设置：

from matplotlib import pyplot as plt  # 导入数据
plt.plot(np.arange(len(df['open'])),df['open'],color = 'red', lw = 2, label = 'open')
# plt.plot(x,y) 通常来讲如果只传入一个参数，那么该参数会被设定为y，而传入数据对应的索引会被自动设定为x
# 参数color可以设定折线的颜色
# 参数lw用于设定折线宽度
# 参数label用于设定数据标签
plt.legend(loc = 1)  # 使用plt.legend()可以显示之前的数据标签，loc参数主要用于设定标签位置
plt.xlabel('Day')  # 设定x轴标签
plt.ylabel('Open')  # 设定y轴标签
plt.title('日开盘价走势图')  # 设定标题
plt.grid(True)  # 开启网格线 默认为Fasle
plt.savefig('Pic01')  # 这个是将图片存储到本地
plt.show()  # 显示图片

np.random.seed(42)  
x = np.random.randn(200)
y = np.random.randn(200)
z = np.random.rand(200) * 9  # 这里事实上生成了一个三维的数据(x,y,z)

plt.figure(figsize=(8, 6))
scatter = plt.scatter(x, y, c = z, cmap='RdBu', alpha=0.7)  # scatter是一个装有所有数据点的容器
# c参数指定颜色参数
# cmap指定色谱图
# alpha指定透明度
plt.title('散点图')
plt.xlabel('1st')
plt.ylabel('2nd')

plt.colorbar(scatter) # 为scatter添加色条图

plt.grid(True, alpha=0.3) 
plt.savefig('Pic10.png')
